# 와이어프레임 업데이트 스킬

`docs/wireframes/` 폴더의 와이어프레임 이미지를 각 페이지 문서에 삽입/업데이트합니다.

## 실행 단계

1. **와이어프레임 파일 확인**
   - `docs/wireframes/` 폴더의 PNG 파일 목록 확인
   - `-labeled` 버전이 있으면 우선 사용

2. **페이지 매핑 규칙**
   | 파일명 패턴 | 대상 문서 |
   |------------|----------|
   | `Landing Page -` | `docs/pages/landing.md` |
   | `Test -` | `docs/pages/test.md` |
   | `Register -` | `docs/pages/register.md` |

3. **와이어프레임 섹션 업데이트**
   - 각 페이지 문서의 `## 와이어프레임` 섹션을 찾음
   - 해당 섹션의 이미지 참조를 업데이트
   - 이미지가 여러 개면 테이블 형태로 정리

4. **이미지 삽입 형식**
   - 단일 이미지: `![Alt](../wireframes/파일명.png)`
   - 복수 이미지: 2열 테이블로 정리

## 실행

위 규칙에 따라 `docs/wireframes/` 폴더를 스캔하고 각 페이지 문서의 와이어프레임 섹션을 업데이트하세요.

새로운 와이어프레임 파일이 추가되었는지 확인하고, 문서에 반영되지 않은 이미지가 있으면 추가합니다.
