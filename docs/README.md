# 소닉 테라피

이른 시간 접속

사용자가 권장된 시각보다 너무 이른 시간에 수면 테라피를 이용할 경우 적절한 이용 시간을 알 수 있다

[https://www.figma.com/design/pNWkeBHQ4OYL6WG6JTrryL/PDT-Line?node-id=8644-2447\&t=pBt7tcKZ75uiEaNu-4](https://www.figma.com/design/pNWkeBHQ4OYL6WG6JTrryL/PDT-Line?node-id=8644-2447\&t=pBt7tcKZ75uiEaNu-4)

### **목적**

* 수면 제한 요법을 적용하는 사용자가 권장된 시각보다 너무 이르게 무조건 이용을 제한하기보다는 적절한 안내를 제공하고자 함
* 다만 너무 늦은 시간에 접속했을 때 제한을 하거나 정보를 줬을 때 사용자의 어노이언스가 높아질 수 있으므로 이른 시간에만 정보 제공
