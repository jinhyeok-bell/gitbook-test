## 1. 페이지 목적

Waiting List 등록 유도 및 수면 테스트로 사용자 흐름 연결

---

## 2. 사용자 플로우

### 플로우 A: 등록 먼저
```
랜딩페이지 접속 → 스크롤 → /#register → 등록 → 테스트 유도
```

### 플로우 B: 테스트 먼저
```
랜딩페이지 접속 → "내 불면증 확인하기" 클릭 → /test → 결과 → 등록
```
