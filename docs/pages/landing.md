|------|------|

| 부작용 | 안전성 강조 |
| 효과 입증 | 인증 및 임상 현황 |

| 효과 시점 | 기대 효과 기간 |
| 비용 | 무료 베타 안내 |

<details>
<summary>Description</summary>

| 질문 | 답변 |
|------|------|
| Sleep Thera는 어떤 원리로 작동하나요? | CBT-I(인지행동치료) 기반의 뇌과학 수면 치료입니다. 불면증의 근본 원인인 과각성(hyperarousal) 상태를 해소하여, 약물 없이도 자연스러운 수면을 회복하도록 돕습니다. |
| 기존 수면제와 어떻게 다른가요? | 수면제는 뇌를 억제해 잠들게 하지만 의존성과 다음 날 멍함, 속 메스꺼움 등 부작용이 있습니다. Sleep Thera는 비약물 방식으로 수면 능력 자체를 회복시켜, 부작용 없이 지속적인 효과를 얻을 수 있습니다. |
| 부작용은 없나요? | 약물을 사용하지 않아 신체적 부작용이 없습니다. EU 의료기기 인증(CE MDR)을 획득하여 안전성이 검증되었습니다. |
| 효과가 입증되었나요? | CE MDR 인증(2025.02), 식약처 혁신의료기기 지정(2025.06)을 받았으며, 서울대병원과 공동 연구를 진행하고 있습니다. 현재 확증 임상시험이 진행 중입니다. |
| 일반 수면 앱과 뭐가 다른가요? | 일반 수면 앱은 소리나 명상을 제공하지만 효과가 임상적으로 검증되지 않았습니다. Sleep Thera는 의료기기로서 임상시험을 통해 치료 효과를 입증한 디지털 치료제입니다. |
| 얼마나 빨리 효과를 볼 수 있나요? | 개인차가 있지만, 당일부터 효과를 느끼시는 분도 있고 7일 이내 적응 기간 후 효과를 보시는 분도 있습니다. 4주 프로그램을 완료하시면 지속적인 수면 개선을 경험하실 수 있습니다. |
| 비용은 얼마인가요? | 현재 베타 테스트 기간으로, 20만원 상당의 4주 프로그램을 무료로 제공하고 있습니다. |

</details>

- 펼쳐진 질문 재클릭 → 답변 접힘
